# Avisail Technologies 事業内容

**最終更新: 2025年12月1日**

## 事業全体構造

Avisail Technologiesは **「貿易 × 投資 × テクノロジー(AI)」** を統合した多層型グローバル企業モデルです。

---

## 1. Technology（投資 × AI × Fintech）

Avisailの基幹事業であり、将来的なキャッシュフローを支えるエンジン。

### ① Aviscore（AIマーケットスコアリング / B2C）

**概要:**
- 多AI合議制エンジンによるマーケット評価システム
- 為替・株・ETFをスコアリング
- リスクラベル、投資判断の理由付け

**技術仕様:**
- UI/UXはSimplyWallSt系の"洗練された金融ダッシュボード"
- サブスクリプションモデル（Stripe導入予定）
- 自動PDFレポート生成（GPT + HTML + PDF化）
- 投資ポートフォリオ分析

### ② ADCS - Active Defense Cyber Security

**概要:**
- Avisailのプロフェッショナル商材として独立した柱
- Active Defense × SOC Framework
- 日本に限らず世界中の中小企業・自治体向け

**技術仕様:**
- GitHubベースのスキャナ
- Shodan / DNS / SSL / Port Scan / Whois / Passive DNS
- 最終レポートは完全自動生成のV1 HTML + PDF
- 章構成: Executive Summary / Attack Surface / DNS … 全10章

**ビジネスモデル:**
- 高粗利・サブスクリプションモデル
- "外部監査 × セキュリティレポート"として販売可能

---

## 2. Global Trading（グローバル商社事業）

**概要:**
- 会社がクイックに動けることを背景にした独立系商社事業
- 中型中古船ブローカー（マッチングモデル）

**主要事業:**
- 船舶売買（小型〜中型のアジアの良質な中古在庫）
- AIエンジンを活用して日々自動更新
- パナマにアカウントを置くことで様々な船舶登録サポートも可能
- エクスクローや船舶購入代金の一部をETF化するなど特徴的な支払い条件

**取り扱い船種:**
- Coaster
- Tugboat
- Landing Craft

**取引地域:**
- 取り扱い: 現金取引が多い中東・アジア（UAE・オマーン・フィリピン etc.）
- 仕入れ: 日本・韓国・東欧
- 販売: 中東・アジア

**財務特性:**
- 借入なし（高キャッシュフロー）
- Panama法人 + Dubai銀行を基盤に運用

---

## 3. Investment（AIによる自動運用や投資に関連するシステム開発）

**概要:**
- AI自動売買・シグナル発信システム（社内資金の投資運用）
- FXの自動売買とETF主体の堅実な資産形成モデル

**技術:**
- AIがリアルタイム分析し、売買アラートやスコア生成
- 将来的にはアプリ化してグローバル展開可能

**ビジネス展開:**
- 将来的には委託を受けて資金運用を実施
- 高収益だが低リスクのキャリートレード戦略
- 高所得者向け

## Corporate Structure（法人構造）

目指している企業構造は以下の三段階:

### ① 日本法人（GK）
- 初期の営業・名刺・公式住所
- 2025年12月に個人事業主として屋号を取得
- 2026年1月に法人設立予定（税金の観点で法人代表と個人事業主を兼任する）

### ② Panama法人もしくは税制の軽いどこか
- 会社のメインアカウント
- 利益の次への投資を税金に左右されることなく最適化する目的
- グローバル取引の中心拠点

### 銀行口座
**Estonia / Dubai（信用 × テック）**
- AI金融系の信用を確保
- Wise口座
- サブスク収益の受け皿
